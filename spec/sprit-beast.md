我直接把"灵兽系统"当成一个完整大模块来设计，能和你前面所有东西（战斗、元素、交易、因果事件）挂上钩。

结构分成这几块：

1. 灵兽在游戏里的定位
2. 捕捉机制（怎么遇到、怎么抓、抓失败有什么后果）
3. 灵兽属性与稀有度（元素、性格、天赋技能）
4. 培养与进化（喂养、训练、突破、契合度）
5. 战斗中的表现形式（出战模式、联动技能、自动行为）
6. 和交易 / 因果系统的联动亮点
7. 一条从"捕捉到成型战斗搭档"的完整示例

## 一、灵兽的定位

灵兽不是单纯"宠物数值加成"，而是：

- **战斗里的第二角色：**
  - 可设置为"自动 AI"，有自己技能与元素。
- **资源生产者：**
  - 采集材料、寻找秘宝、提高掉落。
- **交易影响源：**
  - 稀有灵兽会改变某些材料的产量，从而影响市场价格。
- **因果事件触发器：**
  - 某些事件只在你带着特定灵兽、或灵兽达到某个阶段时出现。

玩家中后期会形成一种流派：

> "我本人是剑修 / 法修，灵兽负责补另一半短板 + 带产能 + 搅动行情"。

## 二、灵兽捕捉机制

### 1. 遇到灵兽的方式

- 野外游历时随机遭遇（按地图/秘境不同，刷不同灵兽池）
- 剧情战后"未完全成形的妖兽幼崽"
- 拍卖行 / 黑市购得"灵兽卵"、"契约残卷"
- 特定世界事件期间（例如"万兽潮"、"灵脉异变"）遇到高稀有度灵兽概率上升

### 2. 捕捉流程（核心是一个小战斗/小博弈）

**基础流程：**

1. 遭遇灵兽 → 进入"灵兽战斗模式"（和普通战斗类似，但目标是"压制"不是"击杀"）。
2. 条件满足时解锁【收服】动作：
   - 比如：灵兽 HP < 30%，且本回合不被秒杀；
   - 或施加了某种"定身/安抚"状态。
3. 玩家选择【投掷契约符 / 灵兽囚笼 / 喂食诱饵】→ 进行一次捕捉判定。

**捕捉判定因素：**

- 灵兽当前 HP 比例（越低越容易）
- 灵兽"好感/恐惧值"临时状态（战中产生：你打得太狠可能恐惧过高反而逃跑）
- 玩家相关技能：
  - 【御兽术】等级
  - 【兽语心法】加成
- 使用的道具品质（高级契约符比破旧符好）
- 灵兽稀有度基础系数（SSS 级很难抓）

可以设计一个文字提示式进度条：

```
【收服进度：■■□】
灵兽在挣扎，似乎还不愿意屈服。
```

**若判定失败：**

- 灵兽有几种反应：
  - 逃跑
  - 暴怒，战斗难度短期上升
  - 或被打残，留下材料但灵兽身死（略有"负因果"的味道）

### 3. 捕捉专用技能树（简单几节点就够）

在"御兽系/神识系"里加几条：

- **【安抚之术】：** 战斗中对灵兽使用，降低其暴躁度，便于捕捉。
- **【兽语心通】：** 捕捉时增加成功率，并在捕捉失败时更容易让灵兽"逃跑而不是拼命"。
- **【灵契强化】：** 你能同时契约的灵兽上限 +1（默认例如 1 只 → 2 只 → 3 只）。

## 三、灵兽属性与稀有度

### 1. 基础数据结构

每只灵兽大致有这些字段：

- `id` / `name`：例如【赤焰狐】【寒泉鹿】
- `rarity`: N / R / SR / SSR / SSS
- `element`: 火 / 水 / 木 / 金 / 土 / 混合
- `role`: 输出型 / 辅助型 / 坦克型 / 采集型 / 混合
- **基础属性：**
  - HP / ATK / DEF / SPD / WIS / SENSE
- **成长率：**
  - 每升一级增加多少属性（决定潜力）
- **性格（性格给加成与减成）：**
  - 【暴躁】：ATK +10%，WIS -5%，训练难度提高
  - 【稳重】：DEF +10%，SPD -5%
  - 【灵动】：SPD +10%，HP -5%
- **天赋技能：**
  - 固定自带+可解锁的一批专属技能
- **特性（类似被动）：**
  - 【灵田之友】：驻守灵田时，药材产量 +X%。
  - 【矿脉啃噬】：驻守矿区时，矿石采集概率 +Y%，但有小概率引发"矿脉骚动"事件。

### 2. 稀有度 & 限制

- **N / R 级：** 普通灵兽，易抓、容易养，主要用于前期或做"生产工具人"。
- **SR 级：** 中高阶战斗灵兽，有一两条不错的天赋。
- **SSR 级：** 核心战斗伴侣，元素与技能都很强。
- **SSS 级：** 传说级，往往需要复杂剧情、特定因果条件才能遇到和收服。

## 四、培养与进化

灵兽培养是一个"长期、带点养成+经营感"的系统。

### 1. 基础培养维度

1. **等级：**
   - 通过战斗获得经验 / 放置训练获得经验。

2. **亲密度 / 契合度：**
   - 通过喂食、陪伴战斗、抚摸互动提高。
   - 契合度越高：
     - 灵兽主动技能效果变强
     - 解锁专属"合击技"
     - 捕捉同类灵兽时成功率+（灵兽帮忙沟通）

3. **心情 / 状态：**
   - 快乐 / 平静 / 焦躁 / 恐惧 / 生病
   - 心情影响：训练效率、战斗表现（暴击率、逃跑概率）
   - 某些道具/事件会改变心情，比如喂喜欢的食物、住在合适的栖息地。

### 2. 喂养系统

- **食物类型：**
  - **普通饲料：** 恢复心情/少量经验
  - **灵果/灵草：** 提升经验 + 少量属性永久加成
  - **珍稀材料：** 可能触发"变异/觉醒"，也可能"吃坏肚子"（风险与收益）

可做一个简单的"喜好"系统：

- 每只灵兽有"喜欢/普通/讨厌"的食物类型：
  - 给喜欢的 → 心情大涨，亲密度提升多
  - 给讨厌的 → 心情降低，甚至拒食

### 3. 训练与突破

**训练方式：**

- **【战斗训练】：** 带去战斗，战斗经验多，但有受伤风险。
- **【静修陪伴】：** 你闭关时，灵兽在旁一同修炼，经验较少但心情稳定。
- **【任务派遣】：** 派灵兽去做采集/押运/巡逻任务，中长期收益+事件。

**突破（类似修士的境界）：**

- 灵兽也有自己的"境界"：凡兽 → 灵兽 → 玄兽 → 圣兽…
- 到达一定等级+亲密度+准备材料后，可以给灵兽突破：
  - **失败：** 暂时虚弱，属性暂时下降。
  - **成功：** 属性大幅提升，获得新的天赋技能/特性。

突破时可写文字：

> "赤焰狐伏在灵阵之中，周身火光翻涌，你以自身灵力为引导，将火铜矿与赤焰丹一并投炉……片刻后，一声清越狐鸣响彻山谷，赤焰狐的气息明显不同于往昔。"

## 五、战斗中的灵兽

### 1. 出战模式

给玩家几个选项：

- **随行出战（默认、副角色）：**
  - 每回合有自己行动条，但可以设置为半自动。
- **召唤出战：**
  - 平时在待命，消耗"召唤值/契约之力"召唤出场，仅战若干回合。
- **辅助出战：**
  - 不直接攻击，只释放辅助技/被动光环。

你可以在设置里给灵兽配置"战斗 AI 模板"：

- **输出型：** 优先使用攻击技能，收割残血
- **坦克型：** 优先用嘲讽、防御技能
- **辅助型：** 优先保护主人、加 Buff

### 2. 灵兽技能类型

- **攻击技：** 按元素造成伤害，可带 DOT / 控制。
- **防御技：** 替主人挡刀、分摊伤害。
- **辅助技：**
  - 加攻击/防御/速度
  - 为主人吸收 DOT
- **合击技（亮点）：**
  - 当"主人 + 灵兽"同时满足条件（气机、元素、亲密度）时解锁：
  - 例如：
    - 剑修 + 火狐：合击【焚风连斩】
    - 水法修 + 冰鹿：合击【极寒冰雨】（大范围减速+伤害）

### 3. 灵兽的战斗成长

- **灵兽每场战斗结束后有自己的战斗总结：**
  - 参战时间、承受伤害、造成伤害、辅助次数
- **某些天赋在累积后觉醒：**
  - 【百战成灵】：参战 X 场后，所有技能冷却 -1。
  - 【不屈之心】：濒死次数到达一定值后，解锁一次"免死"。

## 六、和交易 / 因果系统的联动亮点

### 1. 灵兽影响物资产量 → 影响价格

- **某些灵兽驻守在"灵田 / 矿区 / 药园"：**
  - 让某种材料产量上升 / 品质提高
  - 长期下来，该材料在市场上供给增加 → 价格走弱
- **若你把灵兽派出去"猎捕稀有兽类"：**
  - 短期内某类妖丹供应变多 → 价格下跌
  - 长期可能引发世界事件："某地妖兽几近灭绝"，反向导致稀缺与暴涨。

### 2. 灵兽作为拍卖品 / 债务品

- 灵兽卵 / 契约卷轴可以在拍卖行交易；
- 甚至可以设计"抵押灵兽"：
  - 借灵石时以灵兽做担保，逾期不还被系统或 NPC 回收。

### 3. 因果事件绑定特定灵兽

**举例：**

- **【救下幼崽】事件：**
  - 你选择救一只弱小灵兽，放弃部分战利品。
  - 若多年后该灵兽成长为"圣兽"，会在关键剧情战斗自动来助你一臂之力。
- **【滥杀灵兽】路线：**
  - 你为了材料疯狂猎杀高灵智灵兽。
  - 后续触发"万兽怨潮"类世界事件：
    - 某些地区灵兽极难捕捉/收服；
    - 但相关材料价格暴涨，你的经商能力却大赚一笔 → 典型"短期利益 vs 长期秩序"。

## 七、一个完整的灵兽成长示例（文字演示）

### 1. 邂逅

- 你在"赤焰峡"游历时，遭遇一只【赤焰狐（幼年）】，元素：火。
- 战斗中，你选择用水系法术降低其攻击 + 用身法技能躲避伤害，将其 HP 压到 20%。

### 2. 捕捉

- 使用【安抚之术】降低其暴躁度，再投掷【下品契约符】。
- 提示：
  > "赤焰狐在火光中瑟缩，眼中仍带警惕，但不再疯狂挣扎。"
- 捕捉成功，获得一只 R 级火系输出灵兽。

### 3. 初期培养

- 喂食"火灵草"，它心情提升，契合度+。
- 带它去打低级妖兽，它用小火球帮你补刀。

### 4. 成长 & 突破

- 升到 10 级，你给它准备【火铜矿+赤焰丹】，在"兽栏灵阵"中突破：
- 突破成功，变成【赤焰狐 · 成长体】，获得新技能【狐火缠身】（对单体施加焚灼 + 减防）。

### 5. 战斗合击

- 你是火剑修，点了火元素树的一些节点，亲密度>70。
- 在 BOSS 战中满足条件，解锁合击【焚风连斩】：
  - 你引剑，赤焰狐尾焰加身，剑光与狐火合一，造成超高火伤害 + 多层焚灼。

### 6. 经济与因果

- 中期你把赤焰狐派去"熔岩矿区"驻守：
  - 火铜矿产量明显增加，价格下跌，你可以低价囤货。
- 后面你引导宗门控制矿区 + 在市场操作火铜矿板块：
  - 火系装备成本降低，你和部分宗门弟子战斗中火属性装备更强，形成一个"火系阵营"。
